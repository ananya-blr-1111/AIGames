# AIGames

Daily, tiny HTML-only games (single `index.html` each) generated and committed every day.

## Games

- See `games/` for dated folders.

## How to play

Open any `games/<date>-<slug>/index.html` in a browser.

## Conventions

- Each game is standalone (no build step).
- Uses only HTML/CSS/JS.
- Keyboard controls are described on-screen.
